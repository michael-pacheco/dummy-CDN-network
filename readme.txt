Directions to run the server:

0. Config File as well as "database_server.py" should be checked for their port #'s to ensure no issues happen with unopened ports.
1. Open 3 terminals/shells and go into directory with these files in it.
2. Run "local_DNS_server.py" on one terminal.
3. Run "database_server.py" on one terminal.
4. Run "website.py" on one terminal.
5. Open webbrowser and go to "127.0.0.1:5000", where 5000 is the default port number, but change it if needed.
